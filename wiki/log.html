  • Files updated: wiki/entities/Long-QT-Syndrome.md (source_count 2→3), wiki/entities/Brugada-Syndrome.md (source_count 1→2), wiki/entities/CPVT.md (source_count 1→2), wiki/entities/Short-QT-Syndrome.md (source_count 1→2), wiki/entities/Early-Repolarization-Syndrome.md (source_count 1→2), wiki/entities/Idiopathic-Ventricular-Fibrillation.md (source_count 1→2), wiki/entities/ARVC.md (source_count 4→5), wiki/entities/HCM.md (source_count 3→4), wiki/entities/DCM.md (source_count 1→2), wiki/entities/LMNA.md (source_count 3→4), wiki/concepts/Sudden-Cardiac-Death.md (source_count 3→4), wiki/concepts/Left-Cardiac-Sympathetic-Denervation.md (source_count 1→2), wiki/index.md, wiki/log.md
  • Contradictions noted: ESC 2022 standard ICD threshold for DCM/HNDCM (LVEF ≤35%, Class IIa) contradicts its own multi-risk-factor threshold (LVEF <50% + ≥2 risk factors, Class IIa) — same recommendation class for different thresholds. LMNA-specific ICD threshold inconsistency: ESC CMP 2023 uses LMNA-VTA calculator (≥10%); ESC VA 2022 uses LVEF <50% + ≥2 risk factors (which can include LMNA mutation as one factor) — clinicians must reconcile both tools. CPVT: ICD + flecainide after aborted CA upgraded to Class I in ESC 2022, but ICD + max beta-blocker + flecainide with syncope downgraded from Class I to Class IIa — reflects evolving evidence hierarchy. BrS: catheter ablation for recurrent ICD shocks upgraded to Class IIa from IIb, yet catheter ablation in asymptomatic BrS remains Class III — sharp distinction between symptomatic and asymptomatic that may create ambiguity in intermediate-risk patients.
